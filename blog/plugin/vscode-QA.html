VSCode 常见问题 ​

001_VSCode 中文显示 ​

  • 第一步：找到 扩展，搜索 chinese 或者 访问，点击 Install 进行下载，然后重启 VSCode；

  • 第二步：Ctrl+Shift+P 或者  ⌘ +Shift+P，打开命令行，搜索 language，选择 Configure Display Language，再选择 zh-cn 即可。

    参考资料：完美解决：VSCode 中文显示

002_VSCode 中保存后会在当前目录下自动生成 dist 目录 ​

产生原因：

Sass/Less/Stylus/Pug/Jade/Typescript/Javascript Compile Hero Pro 中文介绍

解决办法：

关闭 compile-hero 插件

  • 在设置中搜索 compile-hero 插件
  • 关闭所有自动生成 dist 目录的选项
如图所示

参考资料：VsCode 中 ctrl+s 后会在当前目录下自动生成 dist 目录

003_VSCode 无法更改文件夹名称 ​

解决办法：

删除文件内node_modules目录，再进行重命名操作。

004_VSCode 卸载删除干净 ​

  • 从控制面板找到 vscode 删除（插件和个人配置）；
  • win+R 输入 %userprofile%，删除当前路径下的.vscode文件夹；
  • win+R 输入 %appdata%，删除当前路径下的code和vscode文件夹即可。
