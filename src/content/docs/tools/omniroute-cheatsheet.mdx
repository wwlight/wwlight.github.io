## 安装

从 `docker run` 转为 `docker compose`，统一到 `~/.docker/compose.yml`。

<Steps>
1. 添加 OmniRoute 服务

    ```yml
    # ~/.docker/compose.yml
    services:
      omniroute:
        image: diegosouzapw/omniroute:main-web
        container_name: omniroute
        ports:
          - "127.0.0.1:20128:20128"
        restart: unless-stopped
    ```

2. 命令

    ```sh
    # 启动
    $ docker compose up -d omniroute
    # 停止
    $ docker compose stop omniroute
    # 更新
    $ docker compose up -d --pull always omniroute
    ```

3. 验证

    ```sh
    $ curl http://localhost:20128/v1/models
    ```

</Steps>

等效 `docker run`（仅作参考）：

```
# docker run -d \
#   -p 127.0.0.1:20128:20128 \
#   --restart unless-stopped \
#   --name omniroute \
#   diegosouzapw/omniroute:main-web
```

## OpenCode 集成

<Tabs syncKey="integration">
  <TabItem label="方式一：Provider 静态配置（推荐）">
    在 `~/.config/opencode/opencode.json` 的 `provider` 字段：

    ```json
    {
      "provider": {
        "omniroute": {
          "api": "openai-compatible",
          "options": {
            "baseURL": "http://localhost:20128/v1"
          },
          "models": {
            "auto/best-free":            { "name": "Auto Best Free", "description": "免费最优，日常首选" },
            "oc/big-pickle":             { "name": "Big Pickle", "description": "OpenCode 免费模型" },
            "oc/deepseek-v4-flash-free": { "name": "DeepSeek V4 Free", "description": "DeepSeek V4 免费版" }
          }
        }
      },
      "model": "omniroute/auto/best-free"
    }
    ```
  </TabItem>
  <TabItem label="方式二：@omniroute/opencode-plugin">
    ```sh title="安装插件"
    $ npm install @omniroute/opencode-plugin
    ```

    在 `opencode.json` 的 `plugin` 字段添加：

    ```json
    {
      "plugin": [[
        "./plugins/node_modules/@omniroute/opencode-plugin/dist/index.js",
        {
          "providerId": "omniroute",
          "displayName": "OmniRoute",
          "baseURL": "http://localhost:20128",
          "modelCacheTtl": 300000,
          "features": { "usableOnly": true }
        }
      ]]
    }
    ```
  </TabItem>
</Tabs>
