整站共用一套主题：**色彩模式（亮/暗）**、**主色（Primary）**、**中性色（Neutral）**、**圆角（Radius）**。在文档站或书签应用顶栏打开主题定制器即可调整；偏好持久化在浏览器 `localStorage`，跨路由与同域标签页生效。

## 可配置项

| 项 | 作用 |
| --- | --- |
| 色彩模式 | 浅色 / 深色 / 跟随系统 |
| Primary | 按钮、链接、强调色（默认 green） |
| Neutral | 背景、边框、灰阶（默认 slate） |
| Radius | 圆角半径（默认 0.25） |

Starlight 文档站与书签应用样式体系不同，但读取**同一套**用户偏好。
