## 本仓库相关路径

| 路径 | 用途 |
| --- | --- |
| `.cursor/rules/` | 项目 Rules（`.mdc`） |
| `.cursor/skills/dev/` | 项目 Skill（`SKILL.md`、`reference.md`） |
| `CURSOR.md` | 仓库内速查 |
| `~/.agents/skills/` | Skills CLI 全局安装（如 `find-skills`、`frontend-design`） |
| `~/.cursor/skills/` | 用户级 Skills |
| `~/.cursor/plugins/` | 已安装 Plugins 缓存 |
