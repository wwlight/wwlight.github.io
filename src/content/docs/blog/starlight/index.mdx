Starlight 文档站相关配置：侧栏、`astro.config.mjs` 组件覆盖、首页 `splash` / `hero` frontmatter。

## 范围

- `sidebar` 与 `src/lib/site-nav.ts` 顶栏模块导航
- `template: splash`、`hero` frontmatter 与 `components.Hero` 覆盖
- MDX 通过 frontmatter 传参，无需 `import Hero`

| 主题 | 路径 |
| --- | --- |
| Starlight 集成 | `astro.config.mjs` |
| 顶栏 / 模块导航 | `src/components/Header.astro`、`src/lib/site-nav.ts` |
| Hero 覆盖 | `src/components/starlight/Hero.astro` |
| 首页内容 | `src/content/docs/index.mdx` |
